# Exercises - Day 12: CSS Styling

## 1. Style Your Resume
Create a simple HTML resume (Name, Experience, Education) and style it using an **External CSS** file.
- Use different fonts for Headings (`h1`, `h2`) and Body text (`p`, `li`).
- Add a background color to the body.
- Center the main heading.
- Add a border around the "Experience" section.

## 2. The Box Model Challenge
Create `3 div` boxes with different background colors.
- Box 1: No padding, No margin.
- Box 2: 20px padding. Notice how the text moves inside.
- Box 3: 20px margin. Notice how it pushes other elements away.
*Use "Inspect Element" in your browser to see the box model layers.*

## 3. Navigation Bar (Flexbox)
Create a horizontal navigation bar with 4 links (Home, About, Services, Contact).
- Use `display: flex`.
- Use `justify-content: space-around` to space them out evenly.
- Remove the bullet points (`list-style: none`).
- Change the color of the link when you hover over it (`:hover`).

## 4. The Button
Create a link `<a>` that looks like a button.
- Give it a background color.
- Add padding (e.g., `10px 20px`).
- Remove the underline (`text-decoration: none`).
- Add `border-radius` to make it rounded.
