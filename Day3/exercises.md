## Basic Level

### Exercise 1: String Length
Write a program to find the length of a string without using the `length()` method.

**Example:**
- Input: `"Hello"`
- Output: `5`

---

### Exercise 2: Character at Position
Write a program to find the character at a given position in a string.

**Example:**
- Input: `"Java"`, position: `2`
- Output: `'v'`

---

### Exercise 3: String Concatenation
Write a program to concatenate two strings without using the `+` operator or `concat()` method.

**Example:**
- Input: `"Hello"`, `"World"`
- Output: `"HelloWorld"`

---

### Exercise 4: Count Spaces
Write a program to count the number of spaces in a string.

**Example:**
- Input: `"Hello World Java"`
- Output: `2`

---

### Exercise 5: Convert Case
Write a program to convert all lowercase letters to uppercase and vice versa.

**Example:**
- Input: `"Hello World"`
- Output: `"hELLO wORLD"`

---

## Intermediate Level

### Exercise 6: Remove Vowels
Write a program to remove all vowels from a string.

**Example:**
- Input: `"Hello World"`
- Output: `"Hll Wrld"`

---

### Exercise 7: Count Words
Write a program to count the number of words in a string.

**Example:**
- Input: `"Java is awesome"`
- Output: `3`

---

### Exercise 8: Reverse Each Word
Write a program to reverse each word in a sentence while maintaining word order.

**Example:**
- Input: `"Hello World"`
- Output: `"olleH dlroW"`

---

### Exercise 9: Check Substring
Write a program to check if a string contains a given substring without using `contains()`.

**Example:**
- Input: `"Hello World"`, substring: `"World"`
- Output: `true`

---

### Exercise 10: Replace Character
Write a program to replace all occurrences of a character with another character.

**Example:**
- Input: `"Hello"`, replace `'l'` with `'p'`
- Output: `"Heppo"`

---

## Advanced Level

### Exercise 11: Longest Word
Write a program to find the longest word in a sentence.

**Example:**
- Input: `"Java is a programming language"`
- Output: `"programming"`

---

### Exercise 12: Check Pangram
Write a program to check if a string is a pangram (contains all 26 letters).

**Example:**
- Input: `"The quick brown fox jumps over the lazy dog"`
- Output: `true`

---

### Exercise 13: Remove Extra Spaces
Write a program to remove extra spaces from a string (keep only single spaces between words).

**Example:**
- Input: `"Hello    World   Java"`
- Output: `"Hello World Java"`

---

### Exercise 14: String to Integer
Write a program to convert a string to an integer without using `Integer.parseInt()`.

**Example:**
- Input: `"12345"`
- Output: `12345`

---

### Exercise 15: Find All Substrings
Write a program to print all possible substrings of a string.

**Example:**
- Input: `"abc"`
- Output: `"a", "ab", "abc", "b", "bc", "c"`

---

## Challenge Level

### Exercise 16: Longest Palindromic Substring
Write a program to find the longest palindromic substring.

**Example:**
- Input: `"babad"`
- Output: `"bab"` or `"aba"`

---

### Exercise 17: String Permutations
Write a program to generate all permutations of a string.

**Example:**
- Input: `"abc"`
- Output: `["abc", "acb", "bac", "bca", "cab", "cba"]`

---

### Exercise 18: Valid Parentheses
Write a program to check if a string has valid parentheses.

**Example:**
- Input: `"()[]{}"`
- Output: `true`
- Input: `"([)]"`
- Output: `false`

---

### Exercise 19: Longest Common Prefix
Write a program to find the longest common prefix among an array of strings.

**Example:**
- Input: `["flower", "flow", "flight"]`
- Output: `"fl"`

---

### Exercise 20: Group Anagrams
Write a program to group anagrams together from a list of strings.

**Example:**
- Input: `["eat", "tea", "tan", "ate", "nat", "bat"]`
- Output: `[["eat", "tea", "ate"], ["tan", "nat"], ["bat"]]`

---

## StringBuilder/StringBuffer Practice

### Exercise 21: Efficient String Building
Write a program to build a string of numbers from 1 to 1000 using StringBuilder.

---

### Exercise 22: Reverse Using StringBuilder
Write a program to reverse a string using StringBuilder methods.

---

### Exercise 23: Delete Characters
Write a program to delete characters from index 5 to 10 in a string using StringBuilder.

---

### Exercise 24: Insert at Position
Write a program to insert a substring at a specific position using StringBuilder.

---

### Exercise 25: Performance Comparison
Write a program to compare the performance of String concatenation vs StringBuilder for 10,000 iterations.
